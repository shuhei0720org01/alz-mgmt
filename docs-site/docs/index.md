# Azure Landing Zones 完全マスターコース

## ようこそ！

この教科書は、Azure Landing ZonesのTerraformコードを完全に理解するためのガイドです。

Azure Landing Zonesを完全に理解して、自分で作れるようになる教材です。

「初心者だけど大丈夫かな...」って思ってますか？  
大丈夫です。このコースは「Terraformって何？」レベルから始まります。


## このコースで学べること

<div class="grid cards" markdown>

-   __Terraformの基本__

    ---

    変数、ローカル変数、モジュールの使い方を基礎から学べます

-   __Azureのリソース__

    ---

    管理グループ、ポリシー、ネットワーク構成を深く理解できます

-   __Landing Zonesの設計__

    ---

    8つの設計領域と実装パターンをマスターできます

-   __全コード解説__

    ---

    リポジトリの全てのコードを詳しく解説しています

-   __環境構築__

    ---

    GitHub ActionsでCI/CDパイプラインまで構築します

-   __トラブルシューティング__

    ---

    よくあるエラーと解決方法を網羅しています

</div>

## 学習のコツ

<div class="grid cards" markdown>

-   __1日30分でもOK__

    ---

    一気に読もうとしないでください。毎日少しずつ進める方が理解が深まります。

-   __コードを読む__

    ---

    実際にコードを読んで理解することが重要です。GitHubのリポジトリのコードを見ながら学びましょう。
    

-   __わからなくても進む__

    ---

    最初は理解できないことだらけです。それが普通。一度全部読んでから、もう一周すると「あ、そういうことか！」ってなります。

-   __メモを取る__

    ---

    自分の言葉でメモを取ってください。「ここがわからない」「これは便利」みたいなメモが後で役立ちます。
