# Azure Landing Zones 完全マスターコース 🚀

## ようこそ！👋

この教科書は、Azure Landing ZonesのTerraformコードを完全に理解して、自分で作って運用できるようになるための教材です。

実際に自分の環境でLanding Zones環境を作成し、手を動かしながら理解を進めていけるように構成しています。

この教科書には最新の技術がたくさんつめこまれているので、読み終えた頃にはクラウドエンジニアとして数段スキルアップしてるはずです✨
